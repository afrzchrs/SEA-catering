1. Untuk dapat menjalankan aplikasi ini membutuhkan aplikasi XAMPP 
2. Kemudian simpan folder SEA-CATERING di dalam htdoc misalnya di C:\xampp\htdocs
3. Kemudian juga simpan juga folder catering(database) di dalam mysql misalnya di C:\xampp\mysql\data
4. Kemudian buka browser dan ketikkan http://localhost/SEA-catering/index.php
5. Jangan lupa untuk menjalankan atau klik Start Apache dan MySQL agar aplikasi dapat terhubung dengan database yang sudah dipasang di folder mysql xampp
6. Aplikasi sudah berjalan dan dapat digunakan.
